如何使用git命令简单地进行操作呢

$ cd your_local_repo
$ git clone git@github.com:lumist/lumist.github.io.git
# 像这个样子就把blog克隆到本地了哦~
# 接着修改本地的html/css/js文件，本地调试完成后开始更新
$ git add *.html
$ git commit -m "commits"
$ git status
$ git push origin master/branches
$ git fetch origin
